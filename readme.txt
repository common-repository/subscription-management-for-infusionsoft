== Description ==

This plugin allows you to create a subscription management page for your Infusionsoft account on your WordPress site.

== Installation ==

To Install:

1. Upload `subscription-management-for-infusionsoft` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the settings (instructions show up after installation)

== Frequently Asked Questions ==

= Will this plugin work out of the box? =

No - you will need to be able to manipulate tags and set up campaigns in Infusionsoft to fully benefit.

= Where can I find documentation on the Infusionsoft setup required? =

Check out [the documentation](https://pirateandfox.com/subscription-management-infusionsoft/)
